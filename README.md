# Fundamentos de Ciencias de Datos — 2026-1

Este repositorio contiene todas las actividades y ejercicios desarrollados en clase para la materia **Fundamentos de Ciencias de Datos**, correspondiente al semestre **2026-1**.

---

## Contenido

A lo largo del semestre se irán agregando actividades relacionadas con:

- Estadística descriptiva (medidas de tendencia central, dispersión y distribución)
- Análisis univariado y bivariado
- Visualización de datos
- Simulación y muestreo aleatorio
- Pruebas estadísticas
